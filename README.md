# 🎓 Student Management System

A modern, lightweight, and fully responsive Student Management System built with vanilla HTML, CSS, and JavaScript. This project helps educational institutions manage student records, track grades, and monitor course enrollments — all stored locally in the browser using LocalStorage.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| ➕ **Add Student** | Add new students with name, email, course, roll number, grade, and status |
| ✏️ **Update Student** | Edit existing student details seamlessly |
| 🗑️ **Delete Student** | Remove students with a single click (with confirmation) |
| 🔍 **Search by Roll Number** | Instantly find any student using their unique roll number |
| 📊 **Dashboard Statistics** | Real-time stats: Total, Active, Inactive, and Average Grade |
| 📚 **Course Management** | Track students by course (B.Tech, B.Sc, BBA, MBA, BCA, MCA) |
| 📈 **Grade Analysis** | View and analyze student grades at a glance |
| 🟢 **Active/Inactive Status** | Mark students as Active or Inactive |
| 🌓 **Light & Dark Theme** | Toggle between Light and Dark modes (saved in localStorage) |
| 📱 **Responsive Design** | Fully functional on desktop, tablet, and mobile devices |
| 💾 **LocalStorage Persistence** | All data is saved locally in your browser |
| 🎨 **Modern UI/UX** | Clean, minimal, and inspired by modern SaaS dashboards |

## 🚀 How to Run

### Option 1: Using Live Server (Recommended)

1. Install **Live Server** extension in VS Code
2. Right-click on `dashboard/index.html`
3. Select **"Open with Live Server"**

### Option 2: Direct Open

1. Simply open `dashboard/index.html` in your browser
2. All data will be stored in your browser's LocalStorage

### Option 3: Landing Page

1. Open `index.html` (root) for the beautiful landing page
2. Click on **"Go to Dashboard"** to enter the application

## 🧪 How to Use

1. **Add Student:** Fill the form and click "Add Student"
2. **View Students:** All students appear in the table below
3. **Edit Student:** Click the edit icon (✏️) next to any student
4. **Delete Student:** Click the delete icon (🗑️) and confirm
5. **Search:** Use the search bar to find students by name, email, or course
6. **Search by Roll Number:** Enter roll number and click search
7. **Toggle Theme:** Click the 🌙/☀️ toggle switch in the top-right corner
